# Administrator Manual

This guide is for the admin.

## Log in

- Log in with the admin account.
- Demo login: `admin@sportshub.local` / `admin123`.

## 1. Approve members

1. Go to **Admin** (Registrations).
2. You see all registrations with their status.
3. Click **Approve** to let a member log in, **Reject** to deny, or **Suspend**
   to disable an active member.
4. The member gets a notification about the change.

## 2. Manage membership categories

1. Go to **Categories**.
2. Click **New Category** to add one, or **Edit**/**Delete** on an existing row.
3. Each category has fees, referral %, withdrawal limit, reward multiplier and
   the facilities it includes.

## 3. View reports

1. Go to **Reports**.
2. You see totals: members, active members, expired memberships, revenue,
   referrals, points issued/withdrawn, category distribution and facility use.
3. Use **Export Excel (CSV)** or **Export PDF (Print)** to save a report.

## Notes

- Only the admin can approve members, manage categories and see reports.
- New members always start as "pending" until you approve them.
